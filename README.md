# Amazon-Style E-Commerce Sales Dashboard (Power BI)

This project is a Power BI dashboard created to analyze Amazon-style e-commerce sales data.  
The goal of this dashboard is to understand **sales performance, customer behavior, and order operations** in a simple and visual way.

> Note: This is a practice project built using sample Amazon-style e-commerce data for learning purposes.
